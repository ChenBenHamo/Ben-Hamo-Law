# יוסי בן חמו - אתר חדש

פרויקט React/Vite לבנייה מחדש של האתר הקיים של יוסי בן חמו משרד עורכי דין.

## הרצה מקומית

```bash
npm install
npm run dev
```

לאחר ההרצה, Vite יציג כתובת מקומית לפתיחת האתר בדפדפן.

אם `npm install` נתקע על `idealTree`, עצרו אותו עם `Ctrl+C` והריצו:

```bash
rm -rf node_modules package-lock.json
npm cache verify
npm install --no-audit --no-fund --progress=false
npm run dev
```

## מה כבר קיים

- דף בית מודרני בעברית וב-RTL.
- עמוד אודותינו עם התוכן הקיים.
- עמוד About us באנגלית.
- עמוד ייעוץ למעסיקים וחברות.
- עמוד מאמרים עם המאמר "מגמה חדשה".
- עמוד פסקי דין עם רשימת המסמכים שהופיעה באתר המקורי.
- עמוד מכתבי תודה עם רשימת ההמלצות מהאתר המקורי.
- עמוד צור קשר עם פרטי משרד, טופס ומפת הגעה.
- כלי נגישות בסיסיים להגדלת טקסט וניגודיות.

## הערות להמשך

עמוד הקישורים באתר המקורי לא נטען בזמן המיפוי, ולכן נשמר עבורו עמוד במבנה החדש עד להשלמת התוכן.
